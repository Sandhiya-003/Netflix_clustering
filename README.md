# 🎬 Netflix Movies & TV Shows Clustering

## 📌 Project Overview

This project focuses on clustering Netflix movies and TV shows based on their features such as type, genre, and rating using KMeans clustering.

## 🎯 Objective

To group similar content and identify patterns that can improve recommendation systems and user experience.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib & Seaborn
* Scikit-learn

## 📊 Steps Performed

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* KMeans Clustering
* Cluster Analysis

## 📈 Key Insights

* Movies dominate Netflix content compared to TV shows
* Drama and Comedy are the most popular genres
* Content production increased significantly after 2015
* Clustering helped identify distinct groups of content

## 🤖 Model Used

* KMeans Clustering

## 📂 Dataset

Netflix Movies and TV Shows dataset

## 🚀 Conclusion

Clustering helps in understanding content patterns and can improve recommendation systems and business decisions.
